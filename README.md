# Manil Rai | Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and the treasures (projects) I've uncovered along my journey as an aspiring web developer.

## About Me

Hi, I'm **Manil Rai**, a 2nd-year BSc (Hons) Computing student at Itahari International College, affiliated with London Metropolitan University. 

I am passionate about:
- Frontend Development
- UI/UX Design
- Building creative web projects

As a big fan of **One Piece**, I've woven a nautical, adventurous theme throughout my portfolio, aiming to bring a touch of imagination into every project I create while maintaining clean and efficient code.

## Built With

- **HTML5 & CSS3**
- **[Tailwind CSS](https://tailwindcss.com/)** (via CDN for rapid styling)
- **JavaScript**
- **Google Fonts** (Inter, Newsreader, Manrope)
- **Material Symbols**

## Features

- **Nautical/One Piece Theme**: Pirate-themed design elements, floating animations, and references to the Grand Line!
- **Responsive Design**: Fully responsive layout catering to desktop and mobile viewers.
- **Dark/Light Mode**: Smooth transitions between themes configured with Tailwind CSS.
- **Project Showcase (Treasures)**: Highlights of my relevant projects including *Travel-Buddy* (Full-stack travel platform) and *CareerLinkPro* (Internship portal).

## Getting Started

To view the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Manilrai/Portfolio.git
   ```
2. Navigate to the project directory and open `index.html` in your favorite web browser. No complex build tools are required.
